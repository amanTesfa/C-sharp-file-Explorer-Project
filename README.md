# C# File Explorer Project

A  ASP.NET Core web application for uploading, previewing, and managing files. Features include:

- Drag-and-drop file upload
- Image/video preview with delete option
- Excel file preview as HTML table
- PDF file preview with embedded viewer
- Download all files as a zip archive
- Modern, responsive UI (Bootstrap)

## Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/amanTesfa/C-sharp-file-Explorer-Project.git
   ```
2. **Navigate to the project folder:**
   ```
   cd C-sharp-file-Explorer-Project
   ```
3. **Run the project:**
   ```
   dotnet run --project FileExplorerProject/FileExplorerProject.csproj
   ```
4. **Open in browser:**
   Visit `http://localhost:5000` (or the port shown in your terminal).

## Features
- Upload images, Excel, PDF, and other files
- Preview images, Excel, and PDF files directly in the browser
- Delete individual files
- Download all files as a zip
- Responsive and modern UI

## Folder Structure
- `FileExplorerProject/` - Main ASP.NET Core project
- `wwwroot/uploads/` - Uploaded files 

## Requirements
- .NET 6 SDK or later
